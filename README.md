## How to Unlock Enterprise Edition
1.  Obtain an official, signed **`LICENSE_KEY`** from the Project Tick Authority.
2.  Place the raw string (e.g. `PT-INTERNAL-PRO-2026-CUSTOMER-xyz.company.com.xxxxxxxxxxxxxxxx`) inside a `LICENSE_KEY` file in the root directory.
3.  The system will perform a cryptographic `openssl_verify()` using our embedded Public Key and strictly authorize your current server's domain.
4.  Once verified, your Admin Dashboard will unlock all restricted modules and switch to **COMMERCIAL / PRO** mode.
